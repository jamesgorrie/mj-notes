What exactly is an isomorphic app ?

Summary

An isomorphic application is one in which uses code which can be run both on the server and on the clients. In the case of a react app, Javascript runs in both the front and backend.

  • React - is defined as an isomorphic library
  • Next - is a framework for the react library

How does it work?

When the user visits the app for the first time, the content is rendered and served by the browser. Each page is rendered by javascript running in the browser.
